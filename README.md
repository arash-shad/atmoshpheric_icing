# atmoshpheric_icing
This repository contains Nek5000 simulation of atmospheric tow-phase flow simulation over a NACA 0012 airfoil. PPICLF library is used to conduct an Euler-Lagarange simulations. More details can be found in the following papers:
[1] Arash Shad, Hashnayne Ahmed, Nadim Zgheib, S. Balachandar, S A. Sherif; Stokes-dependent droplet collection efficiency on a NACA 0012 airfoil from droplet-informed simulations with statistical overloading. Philos Trans A Math Phys Eng Sci 17 July 2025; 383 (2301): 20240368. https://doi.org/10.1098/rsta.2024.0368

[2] Shad, A., Ahmed, H., Zgheib, N., Balachandar, S., and Sherif, S. (September 29, 2025). "The Role of Atmospheric Turbulence on Cloud Droplet Distribution and Droplet Impaction on an Airfoil." ASME. J. Heat Mass Transfer. November 2025; 147(11): 113701. https://doi.org/10.1115/1.4069654
